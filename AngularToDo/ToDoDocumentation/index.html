Angular Documentation Guide

Angular Prerequisite & Installation Commands

Sr. No. Name of Prerequisite Command
1. Install Angular9: npm install --global @angular/cli@9
2. Create new Project / Application: ==> ng new DRF-ToDo-App
==> cd DRF-ToDo-App
3. Generate Component: ==> ng generate component todos
==> ng generate component delete-todo-dialog
==> ng generate component edit-todo-dialog
==> ng generate component todo-item
4. Generate Service: ==> ng generate service djangoapi
5. JS / CSS / Theme: ==> npm install bulma --save
==> npm i --save-dev @types/jquery
==> npm install --save jquery
==> ng add @angular/material
==> npm install @angular/animations --save
6. Install HTTP Request: ==> ng add @angular/http
==> npm install ngx-toastr --save



1. User Registration:

- Just copy and paste this URL: http://localhost:4200 in your browser and you will be see register button. PFA: HomePage or Dashboard

- On click Register button user will regirect to Registration form and fill all the required details to register. PFA: User Registration Form

- After click on Submit, if all the details are fill correctly then you will redirect on login page with registration success message. PFA: Registration Success Screen

2. User Login:

- To open login form, Just click on this URL: http://localhost:4200

- Enter Users Credentails & click on Login button PFA: Login form with Details

- After Successfully login, you will see this Screen

3. ToDos Management ( CRUD on ToDos)

- Click on ToDos tab on top left corner menu & you will redirect to your all ToDos list. How to nevigate on ToDos list

4. Create ToDos

- Now, you can see the form with Todos title, start & end. Fill the form and click on Add button to create new ToDo.

- How to Create ToDo

- Newly Created ToDo

5. ToDos: Mark as Complete

- By default status is Pending.

- Just hove on any particular ToDo which you want to change the status like from Completed to Pending or From Pending to Completed.

- Mark as Complete

- After click on Checkbox, wait for sometime because we are changing the status through API so wait for API response and after success you will see this screen with updated message and status

- Updated with Success status

5. Update ToDo Details

- Just hove on any particular ToDo which you want to update the details & you will see pencil or edit icon on right hand side.

- Edit ToDo

- To Edit the todos data, just click on that Pencil or Edit icon. After click on Pencil icon, you will see your current todos details and now you can able to change the Title, Start & Date according to you and click on Save button & wait for sometime to update using API and after that success response you will see alert with success response and click on Ok and page will automatically refresh with update your ToDos details.

- ToDo: Updated data.

6. Delete ToDo Details

- Just hove on any particular ToDo which you want to delete the details & you will see delete or trash icon on right hand side.

- Delete ToDo

- To Delete that todo, just click on that Delete or Trash icon.

- After click, you will see one delete confirmation pop-up will appear on your screen. If you sure that you want to delete that ToDo then just click on Confirm button else click on Cancel button & wait for sometime to update the API response.

- ToDo: Delete with Confirmation.
